This guide will help you quickly integrate Web LUTs into your project and start applying professional color grading to your images.

## Basic Usage

The simplest way to use Web LUTs is to initialize it and let it automatically process all images on your page:

<Steps>
  <Step title="Import ColorGrader">
    Import the `ColorGrader` class from the package:
    
    ```js
    import { ColorGrader } from 'web-luts';
    ```
  </Step>
  
  <Step title="Initialize with a preset">
    Call `init()` with your desired preset and options:
    
    ```js
    ColorGrader.init({ 
      mode: 'cinematic',
      strength: 0.8
    });
    ```
    
    This will apply the cinematic preset to all `<img>` elements on the page with 80% strength.
  </Step>
  
  <Step title="Done!">
    That's it! Web LUTs will automatically process all existing images and watch for new ones added to the DOM.
  </Step>
</Steps>

<Note>
The `init()` method automatically preloads all preset LUTs in the background for instant switching.
</Note>

## Apply to Specific Images

For more control, you can apply color grading to specific images:

```js
import { ColorGrader } from 'web-luts';

// Apply to a specific element
const heroImage = document.getElementById('hero');
ColorGrader.apply({ 
  target: heroImage, 
  mode: 'vintage',
  strength: 0.9
});

// Apply to multiple elements using a selector
ColorGrader.apply({ 
  target: '.gallery-image', 
  mode: 'cinematic'
});
```

## Dynamic Controls

Web LUTs makes it easy to create interactive controls for your users:

### Adjust Strength

Let users control the intensity of the color grading:

```js
const slider = document.querySelector('#strength-slider');
slider.addEventListener('input', (e) => {
  ColorGrader.setStrength(parseFloat(e.target.value));
});
```

### Switch Between Presets

Allow users to choose different looks:

```js
const radios = document.querySelectorAll('input[name="mode"]');
radios.forEach(radio => {
  radio.addEventListener('change', (e) => {
    ColorGrader.setMode(e.target.value);
  });
});
```

### Reset to Original

Provide a way to remove color grading:

```js
const resetBtn = document.querySelector('#reset-btn');
resetBtn.addEventListener('click', () => {
  ColorGrader.reset();
});
```

## Load Custom LUTs

You can load your own .cube LUT files for complete creative control:

```js
import { ColorGrader } from 'web-luts';

// Load from URL
const customHandle = await ColorGrader.loadCustomCube('/path/to/custom.cube');

// Apply the custom LUT
ColorGrader.apply({ 
  target: 'img', 
  mode: customHandle 
});
```

<Tip>
You can also pass raw .cube file content directly to `loadCustomCube()` instead of a URL.
</Tip>

## Available Presets

Web LUTs includes six built-in presets:

<CodeGroup>
```js normal
ColorGrader.init({ mode: 'normal' });
// No color grading (identity LUT)
```

```js vintage
ColorGrader.init({ mode: 'vintage' });
// Warm, faded film look
```

```js cinematic
ColorGrader.init({ mode: 'cinematic' });
// Hollywood-style teal and orange
```

```js serious-vintage
ColorGrader.init({ mode: 'serious-vintage' });
// Stronger vintage with more contrast
```

```js lights-out
ColorGrader.init({ mode: 'lights-out' });
// Dark, moody look with blue shadows
```

```js sci-fi
ColorGrader.init({ mode: 'sci-fi' });
// Cool blue-green tint
```
</CodeGroup>

## Targeting Specific Images

Use the `selector` option to control which images get processed:

```js
// Only process images with a specific class
ColorGrader.init({ 
  mode: 'cinematic',
  selector: '.graded-image'
});

// Process images in a specific container
ColorGrader.init({ 
  mode: 'vintage',
  selector: '#gallery img'
});
```

## Configuration Options

The `init()` method accepts several options:

```typescript
ColorGrader.init({
  mode: 'cinematic',      // Preset name or custom LUT handle
  strength: 0.8,          // Blend strength (0-1), default: 1.0
  selector: 'img',        // CSS selector for images, default: 'img'
  autoload: true          // Preload all presets, default: true
});
```

## Complete Example

Here's a full example with interactive controls:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Web LUTs Demo</title>
</head>
<body>
  <img id="hero" src="photo.jpg" alt="Hero image">
  
  <div>
    <label>
      Strength:
      <input type="range" id="strength" min="0" max="1" step="0.1" value="1">
    </label>
  </div>
  
  <div>
    <label><input type="radio" name="mode" value="normal" checked> Normal</label>
    <label><input type="radio" name="mode" value="vintage"> Vintage</label>
    <label><input type="radio" name="mode" value="cinematic"> Cinematic</label>
  </div>
  
  <button id="reset">Reset</button>
  
  <script type="module">
    import { ColorGrader } from 'web-luts';
    
    // Initialize
    ColorGrader.init({ mode: 'normal' });
    
    // Strength control
    document.getElementById('strength').addEventListener('input', (e) => {
      ColorGrader.setStrength(parseFloat(e.target.value));
    });
    
    // Mode switcher
    document.querySelectorAll('input[name="mode"]').forEach(radio => {
      radio.addEventListener('change', (e) => {
        ColorGrader.setMode(e.target.value);
      });
    });
    
    // Reset button
    document.getElementById('reset').addEventListener('click', () => {
      ColorGrader.reset();
      document.getElementById('strength').value = 1;
    });
  </script>
</body>
</html>
```
