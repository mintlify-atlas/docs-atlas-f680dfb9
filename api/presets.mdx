## Overview

Web LUTs includes several built-in LUT presets that provide instant visual effects. Each preset is a pre-configured color lookup table that can be applied to images or videos.

## PRESETS Constant

The `PRESETS` constant contains all available preset names:

```typescript
import { PRESETS } from 'web-luts';

console.log(PRESETS.NORMAL);          // 'normal'
console.log(PRESETS.VINTAGE);         // 'vintage'
console.log(PRESETS.CINEMATIC);       // 'cinematic'
console.log(PRESETS.SERIOUS_VINTAGE); // 'serious-vintage'
console.log(PRESETS.LIGHTS_OUT);      // 'lights-out'
console.log(PRESETS.SCI_FI);          // 'sci-fi'
```

## Available Presets

<ResponseField name="PRESETS.NORMAL" type="string" default="'normal'">
  Identity LUT with no color transformation. Use this as a baseline or to reset effects.
</ResponseField>

<ResponseField name="PRESETS.VINTAGE" type="string" default="'vintage'">
  Warm, faded look with boosted reds, slightly reduced greens, and desaturated blues. Creates a nostalgic, retro aesthetic.
</ResponseField>

<ResponseField name="PRESETS.CINEMATIC" type="string" default="'cinematic'">
  Teal shadows with orange highlights. The classic Hollywood color grading look that separates foreground from background.
</ResponseField>

<ResponseField name="PRESETS.SERIOUS_VINTAGE" type="string" default="'serious-vintage'">
  Strong vintage effect with heavily boosted warm tones and significantly desaturated blues. More intense than the regular vintage preset.
</ResponseField>

<ResponseField name="PRESETS.LIGHTS_OUT" type="string" default="'lights-out'">
  Dark, moody atmosphere with blue-tinted shadows. Reduced overall brightness with a cool tone in darker areas.
</ResponseField>

<ResponseField name="PRESETS.SCI_FI" type="string" default="'sci-fi'">
  Cool blue-green futuristic look. Reduced reds with boosted greens and blues for a technological, otherworldly feel.
</ResponseField>

## Using Presets

To use presets, simply pass the preset name to any ColorGrader method:

```typescript
import { ColorGrader, PRESETS } from 'web-luts';

// Initialize with a preset
ColorGrader.init({ mode: PRESETS.VINTAGE });

// Or apply to specific images
ColorGrader.apply({ 
  target: document.getElementById('hero'),
  mode: PRESETS.CINEMATIC 
});

// Or switch presets dynamically
ColorGrader.setMode(PRESETS.SCI_FI);
```

## Preset Values

You can access preset string values directly:

```typescript
console.log(PRESETS.NORMAL);          // 'normal'
console.log(PRESETS.VINTAGE);         // 'vintage'
console.log(PRESETS.CINEMATIC);       // 'cinematic'
console.log(PRESETS.SERIOUS_VINTAGE); // 'serious-vintage'
console.log(PRESETS.LIGHTS_OUT);      // 'lights-out'
console.log(PRESETS.SCI_FI);          // 'sci-fi'
```

<Tip>
  Using the `PRESETS` constant provides autocomplete support and prevents typos. You can also use the string values directly (e.g., `'vintage'`), but the constant is recommended for better developer experience.
</Tip>
