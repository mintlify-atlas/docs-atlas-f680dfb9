## Overview

The `ColorGrader` class is the primary interface for applying LUT (Look-Up Table) color grading to web images. It provides a static API for initializing automatic color grading, applying effects to specific images, and managing color grading state.

## Features

- Apply professional color grading to any `<img>` element
- Supports WebGL 2.0 for high-performance 3D LUTs with fallback to WebGL 1.0
- Built-in presets: `normal`, `vintage`, `cinematic`, `serious-vintage`, `lights-out`, `sci-fi`
- Load custom .cube LUT files
- Control grading strength/intensity (0-1)
- Automatic processing of dynamically added images via MutationObserver
- SSR-safe (no-op on server)

## Import

```typescript
import { ColorGrader } from 'web-luts';
```

## Basic Usage

### Automatic Initialization

Apply color grading to all images at load time:

```javascript
import { ColorGrader } from 'web-luts';

ColorGrader.init({ 
  mode: 'cinematic',
  strength: 0.8
});
```

### Selective Application

Apply to specific images:

```javascript
const heroImage = document.getElementById('hero');
ColorGrader.apply({ 
  target: heroImage, 
  mode: 'vintage' 
});
```

### Dynamic Controls

```javascript
// Strength slider
document.querySelector('#strength-slider').addEventListener('input', (e) => {
  ColorGrader.setStrength(parseFloat(e.target.value));
});

// Mode switcher
document.querySelectorAll('input[name="mode"]').forEach(radio => {
  radio.addEventListener('change', (e) => {
    ColorGrader.setMode(e.target.value);
  });
});
```
