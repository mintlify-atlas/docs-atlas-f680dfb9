## Quick Example

Apply professional color grading to your images with just a few lines:

```js
import { ColorGrader } from 'web-luts';

// Apply to all images at load time
ColorGrader.init({ 
  mode: 'cinematic',
  strength: 0.8
});

// Or apply to specific images
const heroImage = document.getElementById('hero');
ColorGrader.apply({ 
  target: heroImage, 
  mode: 'vintage' 
});
```

<Note>
Web LUTs automatically processes images and maintains the original DOM structure by replacing them with canvas elements.
</Note>

## Built-in Presets

Web LUTs comes with six carefully crafted presets ready to use:

- **normal** - No color grading (identity LUT)
- **vintage** - Warm, faded film look
- **cinematic** - Hollywood-style teal and orange
- **serious-vintage** - Stronger vintage with more contrast
- **lights-out** - Dark, moody look with blue shadows
- **sci-fi** - Cool blue-green tint

<Tip>
You can control the intensity of any preset by adjusting the `strength` parameter from 0 to 1.
</Tip>
